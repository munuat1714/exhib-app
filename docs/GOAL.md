# GOAL.md

## Current Cycle

Cycle 1

---

## Goal

프로젝트의 첫 번째 실행 가능한 Landing Page를 구현한다.

---

## Success Criteria

* Landing Page 생성
* "Exhibition Finder" 제목 표시
* 간단한 소개 문구 표시
* "전시 둘러보기" 버튼 표시
* 모바일 화면 기준 UI
* Component 구조 적용
* npm run dev 오류 없이 실행

---

## Constraints

* React + Vite 유지
* JavaScript 사용
* Tailwind CSS 사용
* 기존 프로젝트 구조 유지
* 새로운 라이브러리 추가 금지
* 서버 코드 작성 금지
* Mock 데이터는 아직 사용하지 않음

---

## Deliverables

Codex는 작업 완료 후 반드시 아래 내용을 제공한다.

* 수정된 파일 목록
* 변경 내용 요약
* 실행 방법
* 다음 Cycle에서 추천하는 작업

---

## Definition of Done

* 애플리케이션이 정상 실행된다.
* UI가 모바일 환경에서 보기 좋게 표시된다.
* 코드에 오류가 없다.
* 다음 Cycle을 진행할 준비가 완료된다.
